# AI Prompts — How to Use

Use these prompts one at a time.

## Best Practice

1. Paste the phase prompt into Claude or Cursor.
2. Do not mix multiple phases.
3. Save the output into the correct folder.
4. Commit to GitHub.
5. Move to the next prompt.
